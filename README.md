PostgreSQL E-Commerce Analytics Project
This project contains four interconnected CSV demo datasets (Customers, Products, Orders, and Order Items), each with 5,000 records and 10 columns, designed for PostgreSQL practice and portfolio development. The repository includes 25 real-world SQL business questions with complete solutions covering joins, aggregations, window functions, ranking, views, customer analytics, product performance, and sales analysis. The datasets simulate an e-commerce environment, enabling users to practice database design, querying, reporting, and data analysis. This project is ideal for SQL learners, data analysts, and anyone building a professional PostgreSQL or GitHub portfolio.
